### 👩🏻‍🎨 `<canvas>` 태그로 만든 재미난 그림판

- `<canvas>` 태그와 **Javascript**로 만든 그림판입니다.
- `그리기`, `채우기`, `지우기`, `초기화`, `저장` 기능을 제공합니다.
- **기본 9가지 색** 외 **랜덤색**과 **사용자 선택 색상**을 지원합니다.
- 약간의 버그가 있긴 하지만 가지고 놀면 꽤 재밌습니다. 🐈💕

## Feature

### Mode

- Draw
- Fill
- Erase
- Clear
- Save
  <br/>

### Color / Brush

- Basic color
- Random color
- User color
- Brush Size
- 현재 선택된 색상 표시
  <br/>

### ETC

- 현재 선택된 모드 버튼 표시
- 모드에 따른 마우스 커서 모양 변경
